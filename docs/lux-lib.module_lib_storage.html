(inner) storage(key, value, config) → {any}

Abstraction over the browser's localStorage API adding the option to provide an alternate solution.

Parameters:
Name Type Description
key string | ResetConfig

The Primary controller of functionality; as a String will indicate which key to interact with in localStorage, as an Object will allow additional functionality.
value any

Any value localStorage will allow for storage.
config string

An object including overrides for default values for: prefix, and store. The prefix value is used to prepend all key- names in the store to ensure against naming collisions withing the store. The store is will enable switching between Session and Local storage as well as other options; also handy for unit testing.

Returns:
  • The value in storage.
Type
any
Example
// set a value in the store
storage('name', 'Joshua');

// retrieve a value from the store
storage('name');

// remove the value from the store
storage({ reset: 'name' });
