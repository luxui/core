HasFunction() → {bool}

Check that the obj (object) prop (array) contains the term (element). Variations - hasAll, hasAny, hasOne - allow for different use-cases with a similar and consistent syntax for searching.

Example
import { hasOne } from './has';

const link = {

hasOne('rel', 'index', link);
