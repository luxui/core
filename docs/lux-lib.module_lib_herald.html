Type Definitions

Herald(arg, arg, …restopt) → {function|true}

A simple pub/sub implementation for decoupled message passing between disconnected components.

Parameters:
Name Type Attributes Description
arg function

a function value will be accepted as a listener ("subscriber") to all messages.
arg any

any value - not a function - will send that as a message to all listners.
rest any <optional>
<repeatable>

any additional arguments will be passed to each listener function as arguments.
Source:
  • lib/herald.js, line 8
Returns:
  • a "remove" function will be returned when a listener is registered to allow unregistering that function, the value "true" will be returned after sending messages to listeners.
Type
function | true
Example
import herald from './herald';

// log any, and all, messages to the console
herald(message => console.log(message));

// log "Hello, world." to the console.
herald('Hello, world.');
